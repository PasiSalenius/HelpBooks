# Installing MyApp

This guide will walk you through installing MyApp on your Mac.

## System Requirements

Before installing MyApp, make sure your system meets these requirements:

- macOS 13.0 or later
- 100 MB of available disk space
- Internet connection for activation

## Installation Steps

1. **Download MyApp**
   - Visit the MyApp website
   - Click the "Download" button
   - Save the installer to your Downloads folder

2. **Run the Installer**
   - Open your Downloads folder
   - Double-click the MyApp installer
   - Follow the on-screen instructions

3. **Launch MyApp**
   - Open your Applications folder
   - Double-click the MyApp icon
   - MyApp will launch and display the welcome screen

## First Launch

When you launch MyApp for the first time:

1. You'll see a welcome screen
2. Click "Get Started" to begin
3. Follow the setup wizard to configure your preferences
4. You're ready to start using MyApp!

## Troubleshooting

### App Won't Open

If MyApp won't open:
- Right-click the app icon and select "Open"
- Go to System Settings > Privacy & Security and allow MyApp to run

### Installation Fails

If the installation fails:
- Make sure you have enough disk space
- Check that you're running a supported version of macOS
- Try downloading the installer again
