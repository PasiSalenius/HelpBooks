# Quick Start Guide

Get up and running with MyApp in just a few minutes!

## Your First Project

1. **Create a New Project**
   - Click the "New Project" button
   - Give your project a name
   - Choose a location to save it
   - Click "Create"

2. **Add Items**
   - Click the "+" button to add items
   - Enter a description
   - Set priorities and deadlines

3. **Organize Your Work**
   - Drag items to reorder them
   - Use tags to categorize items
   - Create folders to group related items

## Essential Features

### The Sidebar

The sidebar shows all your projects and folders. Click any item to view its contents.

### The Toolbar

The toolbar provides quick access to common actions:
- New: Create a new item
- Search: Find items quickly
- Filter: Show only certain items
- Share: Collaborate with others

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| ⌘N | New Item |
| ⌘F | Search |
| ⌘⌫ | Delete Item |
| ⌘, | Preferences |
