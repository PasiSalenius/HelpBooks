# Basic Usage

This guide covers the essential features of MyApp and how to use them effectively.

## Managing Projects

### Creating Projects

Projects help you organize your work into distinct areas:

1. Click **File > New Project**
2. Enter a project name
3. Optionally add a description
4. Choose a color and icon
5. Click **Create**

### Editing Projects

To edit an existing project:

1. Right-click the project in the sidebar
2. Select **Edit Project**
3. Make your changes
4. Click **Save**

### Deleting Projects

To delete a project:

1. Select the project
2. Press ⌘⌫ or choose **Edit > Delete**
3. Confirm the deletion

## Working with Items

### Adding Items

Items represent individual tasks or pieces of work:

```
1. Select a project
2. Click the + button
3. Enter item details
4. Press Return to save
```

### Item Properties

Each item can have:

- **Title**: A descriptive name
- **Notes**: Additional details
- **Priority**: High, Medium, or Low
- **Due Date**: When the item should be completed
- **Tags**: Keywords for categorization

### Completing Items

Mark items as complete by:
- Clicking the checkbox next to the item
- Or pressing ⌘K when the item is selected

Completed items are dimmed but remain visible.

## Search and Filter

### Searching

Use the search bar to find items quickly:

1. Press ⌘F to open search
2. Type your search terms
3. Results appear instantly
4. Press Return to select the first result

### Filtering

Filter items by:
- Priority level
- Due date
- Completion status
- Tags

Access filters from the toolbar or use keyboard shortcuts.

## Preferences

Customize MyApp in Preferences (⌘,):

- **General**: Default project settings
- **Appearance**: Theme and colors
- **Notifications**: Alert preferences
- **Advanced**: Power user features

## Tips and Tricks

- **Drag and Drop**: Reorder items and move them between projects
- **Bulk Edit**: Select multiple items with ⌘-click
- **Quick Add**: Press ⌘N from anywhere to add an item
- **Archive**: Hide completed items with **View > Archive**

For more advanced features, see the [Advanced Features Guide](advanced-features.md).
